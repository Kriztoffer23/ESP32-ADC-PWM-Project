# ESP32 ADC + PWM Project

## Features
- A1: Potentiometer controls LED brightness
- A2: LDR controls automatic light (dark = ON, bright = OFF)
- A3: NTC sensor reads temperature

## Components
- ESP32 DevKit
- Potentiometer
- LDR
- NTC Thermistor
- LED + resistor

## Output
- Serial Monitor shows sensor values
- LED brightness responds in real time
